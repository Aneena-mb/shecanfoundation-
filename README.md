# She Can Foundation Landing Page

A simple and responsive landing page created as part of the Frontend Development Internship selection task for She Can Foundation.

## Features

* NGO logo and branding
* Fixed navigation header
* Login and Register buttons
* Animated welcome message
* Background image
* Mission slogan highlighting women empowerment
* About Us section
* Donate button
* Contact information in footer
* Custom styling with HTML and CSS

## Technologies Used

* HTML5
* CSS3

```text
index.html
README.md
images/
```

## About She Can Foundation

She Can Foundation is an NGO dedicated to empowering women and building a more equal society. The organization supports women through education, training, advocacy, and community programs to create lasting positive change.
